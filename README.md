# Whitewelpe Theme

Ein elegantes, modernes Discord-Theme von [Whitewelpe.de](https://whitewelpe.de).  
Erstellt von **Wölfi-Fe**.

## Installation
1. Lade die `theme.css` Datei herunter.
2. Importiere sie in BetterDiscord unter "Themes" und aktiviere sie.
